## Sobre mim
  Sou estudante de Análise e Desenvolvimento de Sistemas com experiência em modelagem e desenvolvimento de bancos de dados (MER/DER), consultas SQL otimizadas e integração de dados utilizando MySQL e SQLite. Também possuo conhecimento em desenvolvimento de software com Java e Python, aplicando princípios de Programação Orientada a Objetos e boas práticas para criação de sistemas escaláveis.

Tenho experiência com versionamento de código (Git/GitHub), análise e documentação de sistemas com UML, além de participação ativa em projetos acadêmicos, onde atuei em papéis de liderança, coordenando equipes, prazos e apresentações. Isso reforçou minhas habilidades de comunicação, organização e gestão de tarefas.

Também possuo conhecimento em Spring Boot, C# com Unity e já conquistei o título de campeão no Hackathon do MetaversoExp, promovido pela InovaTechJP em parceria com a Prefeitura de João Pessoa.

Objetivo: Busco minha primeira oportunidade de estágio na área de desenvolvimento, onde eu possa aplicar meus conhecimentos, aprender com profissionais experientes e contribuir com soluções de impacto.
