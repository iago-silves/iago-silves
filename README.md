## Sobre mim
  Sou estudante de Análise e Desenvolvimento de Sistemas com experiência em modelagem e desenvolvimento de bancos de dados (MER/DER), consultas SQL otimizadas e integração de dados utilizando MySQL e SQLite. Objetivo: Busco minha primeira oportunidade de estágio na área de desenvolvimento, onde eu possa aplicar meus conhecimentos, aprender com profissionais experientes e contribuir com soluções de impacto.
